# 2015 National Landcover Classification Table

Key for land cover codes and names

## Usage

``` r
lcc2015_codes
```

## Format

A data frame with 19 rows and 2 variables:

- LCC_CODE:

  Land cover code

- LCC_NAME:

  Land cover name
