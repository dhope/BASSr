Dummy hex data to be cleaned

## Usage

``` r
psu_hex_dirty
```

## Format

A spatial data frame with 33 features and 9 fields

- hex_id:

  ID of the hex

- province:

  Province code for that hex

- water:

  Whether that hex is in water or not

- CLC...:

  Land cover columns

- x:

  Geometry
