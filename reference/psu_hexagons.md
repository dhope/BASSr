# Dummy hex data

Dummy hex data

## Usage

``` r
psu_hexagons
```

## Format

A spatial data frame with 33 features and 9 fields

- hex_id:

  ID of the hex

- province:

  Province code for that hex

- water:

  Whether that hex is in water or not

- LC...:

  Land cover columns

- x:

  Geometry
